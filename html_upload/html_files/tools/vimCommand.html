Vim Commands Cheat Sheet

Modes

  • i – Insert before cursor
  • a – Insert after cursor
  • o – New line below
  • O – New line above
  • v – Visual mode
  • V – Visual line
  • Ctrl+v – Visual block

File Commands

  • :w – Save file
  • :q – Quit
  • :wq – Save & Quit
  • :q! – Quit without saving
  • :e filename – Open file

Navigation

  • h – Left, l – Right
  • j – Down, k – Up
  • 0 – Beginning of line
  • $ – End of line
  • w – Next word
  • b – Previous word
  • gg – Top of file
  • G – Bottom of file

Editing

  • x – Delete character
  • dd – Delete line
  • u – Undo
  • Ctrl+r – Redo

Copy & Paste

  • yy – Copy line
  • yw – Copy word
  • p – Paste after cursor
  • P – Paste before cursor

Search & Replace

  • /word – Search forward
  • ?word – Search backward
  • n – Repeat search
  • :%s/old/new/g – Replace all
  • :%s/old/new/gc – Replace with confirm

Useful Tricks

  • . – Repeat last command
  • >> – Indent line
  • << – Unindent line
  • :set number – Show line numbers
  • :syntax on – Syntax highlighting
